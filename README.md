# claude-codex-skills

Claude ve Codex icin paylasilabilir skill koleksiyonu.

## Yapi

Her skill kendi klasorunde tutulur:

```text
skill-name/
  SKILL.md
  references/
```

`SKILL.md` kisa ama yeterli giris dosyasidir.
Uzun standartlar, ornekler ve edge-case notlari ayni skill'in `references/` klasoru altinda tutulur.

## Skilller

- `filament-form-layout`
- `laravel-filament-spatie-media-library`
- `laravel-order-observer-standard`
- `laravel-unique-slug`
